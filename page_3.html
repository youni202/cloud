다이렉트클라우드 소개

삼정데이타서비스의 30년 역사를 바탕으로 한 Microsoft 클라우드 솔루션 제공자

회사 연혁

1992년

삼정데이타시스템 설립, 대한민국 IT 산업의 역사와 함께 성장

1997년

삼정데이타서비스㈜ 법인 전환

2001년

Microsoft Certified Partner (MCP) 등록, Microsoft와 파트너십 시작

2013년

클라우드 전문 브랜드 '다이렉트클라우드' 론칭

2014년

국내 최초 Microsoft Cloud Solution Provider (CSP) 파트너 선정

2016년

메일나라 클라우드 V4.0 GS 인증 1등급 획득

핵심 역량

독보적인 시장 리더십

국내 최초이자 시장점유율 1위의 Microsoft CSP 파트너, 가장 많은 기업 고객이 다이렉트클라우드를 신뢰하고 선택

검증된 전문성

30년 이상의 IT 서비스 업력과 20년간 이어온 Microsoft와의 협업 경험, 국내 최다 Microsoft 365 메일 마이그레이션 경험

프리미엄 기술 지원

체계적인 프리미엄 기술 지원 서비스 제공, Microsoft 공인 커리큘럼 기반의 전문 교육, 24시간 장애 대응 체계

합리적인 가격 정책

Microsoft와의 직접적인 파트너십을 통해 국내 최저가 수준의 경쟁력 있는 가격으로 솔루션 공급

현재: 국내 최다 Microsoft 365 고객사를 보유한 시장점유율 1위 CSP 파트너

3 / 10
